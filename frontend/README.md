# Frontend Code - EventHub

This directory contains the user interface components, assets, pages, and styling files of the EventHub application:

- **`components/`**: Reusable dashboard and layout React components.
- **`app/`**: Route pages (Timeline, My Schedule, Leaderboard, Clubs, and Event Creation).
- **`public/`**: Static public assets.

*Note: For runtime instructions, please refer to the main [README.md](../README.md) at the root of the repository.*
